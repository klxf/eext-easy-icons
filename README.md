# Easy Icons

简单好用易于扩展的图标库扩展

> **Notice:** 使用本扩展需要给予扩展“外部交互”权限，否则无法获取图标数据。 

## 功能介绍

### 便捷使用

安装扩展后按快捷键 `Shift+I` 即可打开扩展界面，在图标库中选择要放置在 PCB 板上的图标，点击画布即可放置

![img_01.png](/images/img_01.png)

### 易于扩展

在扩展设置界面可以添加远程图标库、离线图标包，也可选择扩展收录的公开图标库：

| 扩展设置界面                        | 公开图标库                         |
|-------------------------------|-------------------------------|
| ![img_02](/images/img_02.png) | ![img_03](/images/img_03.png) |

## 创建离线图标包

将 [eIcons 图标库模板](https://github.com/klxf/eicons-repo-template) 下载到本地，按照格式要求配置 `repo.json` 文件、添加图标文件，打包为 `.zip` 压缩包即可

## 创建公开图标库

你可以直接使用模板仓库 [eIcons 图标库模板](https://github.com/klxf/eicons-repo-template) 构建公开图标库
